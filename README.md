# PGGAN
Unofficial custmized PGGAN.

## python PGGAN.py --help
```
usage: PGGAN.py [-h] [--image_dir IMAGE_DIR] [--result_dir RESULT_DIR] [--weight_dir WEIGHT_DIR] [--image_size IMAGE_SIZE] [--lr LR]
                [--mul_lr_dis MUL_LR_DIS] [--batch_size_base BATCH_SIZE_BASE] [--num_train_base NUM_TRAIN_BASE] [--start_depth START_DEPTH] [--cpu]
                [--generate GENERATE] [--noresume]

options:
  -h, --help            show this help message and exit
  --image_dir IMAGE_DIR
  --result_dir RESULT_DIR
  --weight_dir WEIGHT_DIR
  --image_size IMAGE_SIZE
  --lr LR
  --mul_lr_dis MUL_LR_DIS
  --batch_size_base BATCH_SIZE_BASE
  --num_train_base NUM_TRAIN_BASE
  --start_depth START_DEPTH
  --cpu
  --generate GENERATE
  --noresume
```
